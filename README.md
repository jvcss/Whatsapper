## What it is

- Visual app to send message automatically with whatsapp.

## Dependencies

- **PIP Install**
  - Python
  - Selenium
  - Chromedriver
  - Streamlit
  - Regular Expression

## How to play

- Download dependencies

`pip install requirements.txt`

```bash
git clone https://github.com/jvcss/whatsapper.git

cd whatsapper

streamlit run app.py
```
